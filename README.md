# SQL Helper

Extensión de VS Code para insertar snippets SQL, Java JDBC, Python y JavaScript en tus proyectos.  

## Comandos

- `Insertar snippet SQL / Java / Python / JS`: abre un selector de snippets según el lenguaje del archivo activo.

## Instalación

1. Descarga el `.vsix` empaquetado.
2. En VS Code, ve a **Extensiones → ... → Instalar desde VSIX**.
